Bubble sort is a sorting algorithm that works by comparing 2 ascii values of the 1st 2 characters in the words, if one of the values is greater then it stores the string and compares it against the next one until it finds the biggest one and has it brought to the top. If it were to find 2 characters identical then it would move onto the second character and compare the same way all the way until it finds a difference. Insetion Sort works by starting at the beginning incrimenting up and as soon as a smaller ascii value is found it is brought back down through the same numbers until it finds its place. Selection sort works by finding the smallest ascii value in the entire array and then moving it down to the bottom then the next smallest is moved one above the bottom and this continues throughout the entire array. The underlying reason for these differences in these sorting algorithms is because different types of collected data is better suited to some sorting methods.
